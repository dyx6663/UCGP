# Security Policy

UCGP is released for authorized robustness auditing and academic research on infrared vision-language systems.

Please do not use this repository to evaluate, probe, or attack systems without explicit permission from the system owner.

## Reporting Issues

If you find a bug that affects reproducibility, safety, or the responsible use of this code, please open a GitHub issue with:

- the command or configuration that triggered the issue,
- the expected behavior,
- the observed behavior,
- relevant environment details such as Python, PyTorch, CUDA, and MMDetection versions.

Do not include private datasets, credentials, access tokens, or confidential model checkpoints in public issues.
